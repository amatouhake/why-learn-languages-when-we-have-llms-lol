## 中優先度

### 音声読み上げの速度変更
単語の音声再生速度を調整できるようにする。

- `Audio.playbackRate` で 0.5x〜1.5x 程度の速度変更
- セットアップ画面 or クイズ画面にスライダー/ボタン
- localStorage で設定を永続化

### 回答後に正解の読み上げ
不正解時に正解の単語の音声を自動再生。正しい発音をすぐ確認できる。

- 既存の `/audio/cmn-{simplified}.mp3` を活用
- 正解時は現状通り（出題時に再生済み）、不正解時のみ追加再生

### セッション統計の表示
クイズ終了（ホームに戻る）時に今回のセッション結果サマリーを表示。

- 正答率、回答数、平均速度、苦手だった単語リスト
- モチベーション維持に効果的

### HSK 4-6 対応
現在 HSK 1-3（595語）のみ。上級レベルの追加。

- `complete-hsk-vocabulary/` に HSK 4-6 のデータがあるか確認
- `llm_generated.json` の日本語訳・例文も追加生成が必要
- 音声ファイルの追加調達

## 低優先度

### 例文 TTS 音声
例文を音声で読み上げる。

- 595文 × 外部 TTS API or 事前生成が必要
- Pi のストレージ制約あり（現状の単語音声で ~50MB）
- 単語音声は既にあるので学習効果の上積みは限定的
- やるなら Web Speech API（ブラウザ内蔵 TTS）が低コスト

### キーボードショートカットの拡充
- スペースキーで次の問題（手動遷移モード実装時に対応）
- `R` キーで音声リプレイ
- `Esc` でホームに戻る

### ダークモード / ライトモード切替
現在ダークテーマ固定。好みに応じて切替。

- `prefers-color-scheme` メディアクエリ or 手動トグル
- 優先度低：個人用途でダークで問題なければ不要

## 実装済み

- **苦手な単語の重点出題** — `quiz.py` の `_select_weighted_word()` で実装済み。未回答: 5倍、正答率80%未満: 3〜10倍、習得済み: 1倍のウェイト
- **効果音** — Web Audio API 合成トーン（正解: 上昇、不正解: 矩形波ブザー）+ トグル
- **PWA 対応** — manifest.json + Service Worker。静的アセット cache-first、音声 lazy cache、API network-only。ホーム画面追加でスタンドアロン起動
- **手動遷移モード** — 回答後スペース/タップ/数字キーで次へ。遷移モード4段階（手動/自動/高速/即時）、localStorage 永続化
- **例文表示改善** — 手動遷移で解決。例文エリアにアクセントボーダー追加
- **設定の永続化** — HSKレベル・言語ペア・出題方向・音声を localStorage に保存
- **モバイルレイアウト** — max-height ベースのメディアクエリ
